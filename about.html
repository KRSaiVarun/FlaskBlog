About FlaskBlog

Welcome to FlaskBlog

FlaskBlog is a simple yet powerful blogging platform that allows you to share your thoughts, ideas, and stories with the world. Despite its name, FlaskBlog is built with Django, a high-level Python web framework that encourages rapid development and clean, pragmatic design.

Features

  • Create and Edit Posts

    Easily create, edit, and delete your blog posts.

  • Modern Design

    Clean and responsive design that looks great on all devices.

  • User Authentication

    Secure login system to protect your content.

  • Social Sharing

    Share your posts on popular social media platforms.

Technology Stack

This blog is built with these technologies:

Python Django Bootstrap 5 SQLite JavaScript HTML5 & CSS3
